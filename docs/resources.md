# 推荐资料

这些链接用于补充阅读和实验参考。数量先保持少，后续可以按章节继续细分。

## 教材与讲义

- [Operating Systems: Three Easy Pieces](https://pages.cs.wisc.edu/~remzi/OSTEP/)：免费在线操作系统教材，主线是虚拟化、并发和持久化，适合作为概念复习的外部参考。
- [MIT 6.1810: Operating System Engineering](https://pdos.csail.mit.edu/6.1810/2025/schedule.html)：MIT 操作系统工程课程，包含课程安排、讲义、作业和实验入口。
- [xv6: a simple Unix-like teaching operating system](https://pdos.csail.mit.edu/6.828/xv6)：xv6 教学操作系统主页，适合结合启动、系统调用、页表、陷入和调度内容阅读源码。
- [The Linux Kernel documentation](https://docs.kernel.org/)：Linux 内核官方文档，适合在学完基础概念后查阅真实内核中的 API、子系统和架构说明。

## 使用建议

- 先用本站笔记建立章节框架，再用 OSTEP 补足概念解释。
- 做实验或看源码时优先查 MIT 6.1810 和 xv6 资料。
- Linux 官方文档更适合验证真实系统实现，不建议在第一次复习时从头读。
