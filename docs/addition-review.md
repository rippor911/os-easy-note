# 补充综合测试

这一页把 `local_temp/addition.md` 中整理出的易错题汇总成一套补充测试，当前已经覆盖内存管理、进程基础，以及新补进来的同步与互斥错题，适合在复习完相关章节后集中回查。

## 使用说明

- 单选、多选题可直接点选后提交。
- 部分原题的完整选项文本没有保留，只能按老师原题的选项字母回忆作答；这些题在页面里已单独标注。
- 每题提交后立即显示标准答案和解析，同时会进入本地做题/错题记录。
